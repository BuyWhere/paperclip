# v2026.720.1

> Released: 2026-07-20

## Highlights

- **GET /api/agents/me no longer exposes adapter secrets in plaintext** — The `adapterConfig.env` binding values for both `plain` and `secret_ref` types are now redacted to `{type, present: true}` in the response, preventing token and credential leakage to unauthorized callers. ([BUY-59809](https://github.com/BuyWhere/paperclip/issues/59809))

## Fixes

- **Release workflow uses correct NPM token for publish jobs** — Both `publish_canary` and `publish_stable` jobs now use `NPM_TOKEN` environment secret directly instead of the broken OIDC token-gon action, restoring automated canary and stable npm publishing. ([BUY-63611](https://github.com/BuyWhere/paperclip/issues/63611))
- **PAT-pattern redaction now covers underscore-delimited Personal Access Token suffixes** — Regex updated to match `_PAT` variants (e.g., `GH_PAT_TOKEN`, `POSTHOG_PAT`) so these high-privilege tokens are properly redacted in event logs. ([BUY-59809](https://github.com/BuyWhere/paperclip/issues/59809))
